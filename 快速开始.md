## 📋 详细步骤

### 1. 检查 Docker Desktop

打开 Docker Desktop，确认系统托盘图标显示"运行中"。

### 2. 检查 OpenClaw 镜像

打开 PowerShell，运行：
```powershell
docker images | findstr openclaw
```

应该看到类似输出：
```
openclaw    latest    abc123def    2 hours ago    500MB
```

### 3. 首次运行

如果是第一次运行，会自动：
- 安装 Node.js 依赖
- 安装 React 依赖
- 创建配置文件
- 检测 Docker 和镜像

### 4. 后续运行

只需双击 `启动应用.bat` 即可！

### 常见问题

**Q: Docker 未运行？**
A: 启动 Docker Desktop，等待图标显示"运行中"

**Q: 找不到镜像？**
A: 编辑 `.env` 文件，设置正确的镜像名称

**Q: 端口被占用？**
A: 编辑 `.env`，修改 PORT 和 CLIENT_PORT
